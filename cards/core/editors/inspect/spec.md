# Inspect

A tool you pin onto an embed to see what's behind it: the package that draws it,
and — when it's showing a document — that document too, rendered side by side.

- a debugging aid for the [canvas](automerge:2inNEjqtyHgqxUHAphpoiHvaM2s6), not an
  everyday card
- resolves both the package and the document straight from the embed
