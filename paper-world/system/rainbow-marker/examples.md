# Rainbow Marker

## Rainbow wave

A gentle wavy rainbow stroke.

```json
{
  "tool": "rainbow-marker/tool.json",
  "tags": [],
  "value": {
    "x": 0, "y": 0, "viewUrl": "rainbow-marker/tool.json",
    "points": [[10, 50, 0.5], [30, 30, 0.6], [55, 55, 0.7], [80, 25, 0.6], [110, 50, 0.55], [140, 30, 0.65], [170, 55, 0.5], [200, 35, 0.6]]
  },
  "width": 220,
  "height": 80
}
```

## Rainbow marker button

The toolbar button that activates the rainbow marker drawing tool.

```json
{
  "tool": "paper/tool.json",
  "tags": [],
  "value": {
    "shapes": {
      "btn": { "x": 10, "y": 10, "viewUrl": "rainbow-marker/button.json" }
    }
  },
  "width": 200,
  "height": 80,
  "create": "shapes.btn"
}
```
