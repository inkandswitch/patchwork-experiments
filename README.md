# Patchwork Tools

A repository of the core and best supported (relatively speaking) Patchwork tools.

This repository should only include Patchwork 2.0 (-next / tiny) tools. For past tools see `patchwork-extra` or `patchwork-legacy`.

## Guidance
Each tool should be independently buildable without any dependencies on other folders. This is important to avoid creating accidental dependencies between tools and allow independent compilation and distribution
