# Rectangle

## Small square

A compact 100x100 square.

```json
{
  "tool": "rectangle/tool.js",
  "value": { "x": 0, "y": 0, "toolUrl": "rectangle/tool.js", "width": 100, "height": 100 },
  "width": 100,
  "height": 100
}
```
