# Embed

## Embed button

The toolbar button that opens the embed type menu and activates the embed placement tool.

```json
{
  "tool": "paper/paper.js",
  "value": {
    "shapes": {
      "btn": { "x": 10, "y": 10, "toolUrl": "embed/button.js" }
    }
  },
  "width": 200,
  "height": 80,
  "create": "shapes.btn"
}
```
