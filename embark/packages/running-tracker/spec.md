# Run Tracker

A mobile running tracker: start a run and it records your route and pace from the
phone's location, then files it into a log of past runs with distance, time, and
stats.

- one log gathers every run; each run is its own document
- open the log for the full app, or a single run for its summary
- recorded runs can be linked from notes with an
  [@mention](automerge:WHoh3hX59pHz81BY59VkhXEhzRD)
