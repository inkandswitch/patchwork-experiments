# LLM Card

Write a plain-language description of an effect you want, hit Activate, and a
language model writes a small program for it — which then runs live on the
[canvas](automerge:2inNEjqtyHgqxUHAphpoiHvaM2s6), using the same shared state
every other card does.

- the generated code lives in the card's own folder and can be re-run
- it also writes a plain-language spec of what it built
- a transcript shows the model's work as it goes
