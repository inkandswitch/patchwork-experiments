# Mentions

Adds `@mentions` to text editors: type `@` to open a [search](../core), then pick
a result to drop in a live link to that document. The link always shows the
document's current title, and clicking it opens the document.

- works in any note or essay
- results come from whatever is listening — like the
  [Mention Finder](automerge:2qhWc5S2pg83z2xutpiCkafYkdSN)
- a focused link and the document it points to light up together
