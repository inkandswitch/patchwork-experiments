# Metric Converter

A card that reads your notes and annotates every metric quantity with its
imperial equivalent, shown as a
[sticker](automerge:2fygfMPPm6BdmtgRXbpQvVBbizRL) right after the value. It's the
mirror of the [Unit Converter](automerge:2YXL4FwZ7crmDpgcm2FobPGpQyE7) and is a
fully standalone card — it shares no scanning code with it.

- handles km, m, kg, and °C
- converts running pace written `M:SS /km` into `M:SS /mi` (e.g. `6:24 /km` →
  `10:18 /mi`), the headline case for a pasted run summary like
  `32:10 · 6:24 /km`
- updates as you type
