# Timer

A card that turns timer tokens in your notes into live countdown widgets you can
start and reset, each rendered in place as a
[sticker](automerge:2fygfMPPm6BdmtgRXbpQvVBbizRL).

- write `@timer 5m` or a bare `5:00`
- the countdown widget itself comes from
  [stickers](automerge:2fygfMPPm6BdmtgRXbpQvVBbizRL)
