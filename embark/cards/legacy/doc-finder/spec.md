# Mention Finder

A card that answers [@mention](automerge:WHoh3hX59pHz81BY59VkhXEhzRD) searches: as
you type, it offers the documents already nearby whose title matches — so you can
link to runs, notes, and anything else from inside a note.

- matches on each document's title, case-insensitive
- only offers documents that are reachable around it
