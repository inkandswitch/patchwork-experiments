# Slash Commands

Adds a `/` menu to text editors: type `/` to see the commands on offer, then pick
one to drop its result into the note. The commands come from cards like
[Weather](automerge:2gtsy4b6hU38DQAMPk6kYHLwxrxE) and
[Routes](automerge:41HBbYkbrqYd9STaojjQUsFc1jDW).

- each suggestion previews as the real thing it will insert, not just a label
- works in any note or essay
- does nothing until a card offers commands
