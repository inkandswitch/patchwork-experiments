# Counter

## Fresh counter

A counter in its default state.

```json
{
  "tool": "counter/tool.json",
  "value": { "count": 0 },
  "width": 200,
  "height": 100
}
```

## Counter at 42

A counter with a non-zero value.

```json
{
  "tool": "counter/tool.json",
  "value": { "count": 42 },
  "width": 200,
  "height": 100
}
```

## Negative counter

A counter that has been decremented below zero.

```json
{
  "tool": "counter/tool.json",
  "value": { "count": -7 },
  "width": 200,
  "height": 100
}
```
