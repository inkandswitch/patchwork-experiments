# Stack

## Default stack

A stack with a paper canvas and dock layout layered together.

```json
{
  "tool": "stack/tool.json",
  "value": {
    "stack": {
      "children": [
        { "viewUrl": "paper/tool.json" },
        { "viewUrl": "dock-layout/tool.json" }
      ]
    }
  },
  "width": 400,
  "height": 300
}
```
